ΘΕΜΑ Β1 Στάσιμο κύμα – χορδή με ακλόνητο άκρο

✓ Σωστή απάντηση: (iii)
Αρχική κατάσταση (3 δεσμοί μαζί με το ακλόνητο άκρο):
L = λ1/4 + λ1/2 = 3λ1/4    (1)

Τελική κατάσταση (2 δεσμοί μαζί με το ακλόνητο άκρο):
L = λ2/4 + λ2/2 + λ2/2 = 5λ2/4    (2)

(1), (2) ⇒ 3λ1/4 = 5λ2/4 ⇒ 3λ1 = 5λ2
λ = v/f = v·T ⇒ 3(v·T1) = 5(v·T2) ⇒ 3T1 = 5T2 ⇒ T1/T2 = 5/3
Άρα T1/T2 = 3/5 → αντιστοιχεί στην επιλογή (iii)

ΘΕΜΑ Γ Φαινόμενο Compton – Φωτοηλεκτρικό φαινόμενο

Γ1. λ' - λ = (h/me·c)(1 - συνφ) με φ = 180°, λ = 8λc:
λ' - 8λc = λc(1 - συν180°) = λc(1+1) = 2λc ⇒ λ' = 10λc

Γ2. Eφ = hc/λ = hc/(8λc) = me·c²/8, E'φ = hc/λ' = me·c²/10
ΑΔΕ: Ke = Eφ - E'φ = me·c²(1/8 - 1/10) = me·c²/40
me·c² = 511 keV = 5,11×10⁵ eV ⇒ Ke = 5,11×10⁵/40 = 1,2775×10⁴ eV
Ke = 1,28×10⁴ eV

Γ3. Από εξίσωση Einstein: Ke = hf - φ. Για Ke = 0: hf0 = φ ⇒ f0 = φ/h
φ = 1,4 eV = 1,4×1,6×10⁻¹⁹ J = 2,24×10⁻¹⁹ J, h = 6,4×10⁻³⁴ J·s
f0 = (2,24×10⁻¹⁹)/(6,4×10⁻³⁴) = 0,35×10¹⁵ Hz = 3,5×10¹⁴ Hz
f0 = 3,5×10¹⁴ Hz

Γ4. Ke = hc/λ - φ = 1240 eV·nm / 400 nm - 1,4 eV = 3,1 eV - 1,4 eV = 1,7 eV (≈1,6 eV από εκφώνηση)
Τάση άρσης: V0 = Ke/e = 1,6 V
V0 = 1,6 V
Σχήμα Γ: Σκεδαζόμενο φωτόνιο και ανακρουόμενο ηλεκτρόνιο
